A web-based VIN decoder that allows users to decode single or multiple VINs and view detailed vehicle information such as specifications, safety ratings, recalls, and complaints using NHTSA APIs.

The app includes user authentication, credit-ready architecture, and smooth Framer Motion animations for an engaging UI experience. Built with React + Vite and deployed on Cloudflare.

# Tech Stack
- Frontend: React + Vite
- Auth: Firebase Authentication
- APIs: NHTSA Vehicle APIs
- Hosting: Cloudflare Pages
